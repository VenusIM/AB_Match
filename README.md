# AB_Match
구인, 구직자 간의 니즈를 파악하여 매칭 서비스 프로젝트

### #Introduce
어머니 가게를 도와드리며 구인의 어려움을 겪어 시간적 여유가 없는 사람들을 위해 원하는 일터, 인재를 자동으로 매칭해주는 서비스를 제공할 계획입니다.
